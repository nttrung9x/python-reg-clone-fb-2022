# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/)

```bash
pip install ppadb
pip install pyzbar
pip install pyotp
```

## Usage

```python
RegClone.py
```
